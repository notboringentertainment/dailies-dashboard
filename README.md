# Dailies Dashboard

AI project file dashboard — scan folders, view metadata, filter/search, generate rename plans.

## Quick Start

```bash
npm install
npm run dev
```

Then enter a folder path in the scan bar and click Scan.

## Features

- Recursive folder scanning with ffprobe metadata extraction
- Grid and table views with search and multi-filter
- Rename plan generator with CSV export
- Dark/light mode
- Export filtered results to CSV
